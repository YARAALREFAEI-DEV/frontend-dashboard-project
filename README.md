# Frontend Dashboard Project

##  Overview
This project is a responsive Frontend Dashboard developed during my internship training with a software company.  
It was built from scratch using core web technologies to simulate a real-world admin dashboard system.

##  Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design (Flexbox / Grid)

##  Features
- Fully responsive admin dashboard layout
- Sidebar navigation menu
- Interactive UI components
- Clean and modern design
- Structured layout for real-world use cases

##  My Role
I independently designed and developed the entire frontend dashboard from scratch during my training period, including UI structure, styling, and interactivity.

##  Purpose of the Project
This project was created as part of my internship to strengthen my frontend development skills and understand real-world UI/UX dashboard structures used in web applications.
